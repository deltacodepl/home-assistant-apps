Expression Helpers
==================

For generating meaningful values with expressions, you usually need access
to the state of entities in Home Assistant and other data, such as the
current date and time. Several functions and variables are available for
usage in your expressions, these will be described in this chapter.

.. note::

   Depending on the actor type you're using, there may be additional
   helpers available. See the documentation of the particular actor type.
