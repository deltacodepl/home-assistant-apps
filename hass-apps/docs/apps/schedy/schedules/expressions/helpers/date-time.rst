Date and Time
=============

``datetime``
------------

``datetime: ModuleType``

Python's ``datetime`` module.


``now``
-------

``now: datetime.datetime``

A ``datetime.datetime`` object containing the current date and time.


``date``
--------

``date: datetime.date``

A shortcut for ``now.date()``.


``time``
--------

``time: datetime.time``

A shortcut for ``now.time()``.
